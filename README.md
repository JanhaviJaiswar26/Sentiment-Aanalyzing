## Methods
- Preprocessing: HTML cleaning, stopword removal, Porter stemming
- Feature extraction: CountVectorizer (1000 features)
- Models: GaussianNB, MultinomialNB, BernoulliNB

## Results
- MultinomialNB: 82.47% accuracy, 82% precision
- GaussianNB: 78.86% accuracy
